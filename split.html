Split PDF File

Extract pages from your PDF document

Enter page numbers separated by comma. Leave empty to download every page as a separate file.

How to Split PDF Pages Online?

  1. Upload your PDF file using the file selector above.
  2. Enter the page numbers you want to extract (e.g., "1, 5, 10").
  3. If you want to separate ALL pages into individual files, leave the box empty.
  4. Click the "Split PDF Now" button.
  5. Your selected pages will download instantly to your device.

Secure & Private Splitting

Security is our top priority. This Split PDF tool runs 100% in your browser using advanced JavaScript libraries.

  • Your file is never uploaded to our server.
  • No one else can see your document.
  • It works even if you disconnect from the internet after loading the page.
